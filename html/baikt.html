What is Ajax?

As defined in Wikipedia

"AJAX is a group of interrelated web development techniques using many web technologies on the client side to create asynchronous web applications. With Ajax, web applications can send and retrieve data from a server asynchronously (in the background) without interfering with the display and behavior of the existing page.
